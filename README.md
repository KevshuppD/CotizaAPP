# CotizaAPP

Cotizador dinámico de derechos de sepultación.

## Funcionalidades
- Cálculo de cuotas en CLP y UF.
- Gráfico dinámico de capacidad de sepultura.
- Exportación de cotización como imagen.
- Selección de parque.
- Valores de UF y fechas dinámicos.

## Despliegue en Vercel
Este es un sitio estático. Simplemente conecta este repositorio a Vercel o arrastra la carpeta a su panel de control.
